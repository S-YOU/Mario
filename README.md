# Mario
A mario 1985 version, in Smalltalk

## Installation on Pharo
Execute the following incantation in a Playground:

```Smalltalk
Metacello new
    baseline: 'Mario';
    repository: 'github://akevalion/Mario';
    load.
``` 

To run the game execute 
```Smalltalk
MaGame new run
```
